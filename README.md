# Installation Guide

### Composer
Run: ```composer instal```

### Npm
Run: ```npm run build``` or ```yarn install```

### Generate key
Run: ```php artisan key:generate```


### Migrations and seeders
Run: ```php artisan migrate``` or ```php artisan migrate:fresh```

Run: ```php artisan db:seed```

## Ziggy
Run: ```php artisan ziggy:generate resources/js/src/ziggy.ts```

## Build
Run: ```yarn build``` or ```npm run build```
